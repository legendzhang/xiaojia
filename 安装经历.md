## 📋 安装步骤

### 1. 环境准备

```bash
# 检查 Node.js 版本
node -v

# 检查 Git 版本
git --version
```

### 2. 克隆项目

```bash
git clone https://github.com/legendzhang/xiaojia.git
cd xiaojia
```

### 3. 安装依赖

```bash
npm install
```

### 4. 配置

```bash
cp .env.example .env
# 编辑 .env 文件配置相关参数
```

### 5. 运行

```bash
npm start
```
