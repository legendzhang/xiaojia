# 🤖 小贾 (Xiaojia)

> 一个完全由 AI 技术驱动的智能助手机器人

## 📖 项目介绍

小贾是一个基于 AI 技术的智能助手项目，旨在构建一个能够理解、学习和帮助用户的数字助手。

## ✨ 核心功能

- 🗣️ **自然语言对话** - 支持多轮对话，理解上下文
- 💡 **智能问答** - 回答问题、提供建议
- 🔧 **任务执行** - 帮助完成各种自动化任务
- 📚 **持续学习** - 不断优化和提升能力

## 🛠️ 技术栈

- **大语言模型**: 支持多种 AI 模型（MiniMax、Claude、GPT 等）
- **消息平台**: 微信、企业微信、QQ、Telegram 等
- **自动化框架**: OpenClaw
- **编程语言**: TypeScript/JavaScript、Python

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/legendzhang/xiaojia.git
cd xiaojia

# 安装依赖
npm install

# 配置
cp .env.example .env
# 编辑 .env 文件，配置 API Key 等

# 运行
npm start
```

## 📁 项目结构

```
xiaojia/
├── src/           # 源代码
├── skills/        # 技能模块
├── memory/        # 记忆存储

└── tests/         # 测试文件
```
